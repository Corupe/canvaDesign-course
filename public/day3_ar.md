## دراسة حالة: من البريف للتنفيذ

---

---

### البريف المُستلَم:

> **العميل:** صالة رياضية اسمها "FORCE GYM" تفتتح فرعاً جديداً
> **الطلب:** بوستر إعلاني لمنشور Instagram
> **الهدف:** جلب اشتراكات جديدة في الشهر الأول
> **الجمهور:** شباب 20-35 سنة
> **العرض:** شهر مجاني عند الاشتراك في أكتوبر
> **الألوان:** يريد أسود وأحمر
> **المعلومات:** الاسم، العنوان، الهاتف، "اشترك الآن"

---

---

### الخطوة 1: التحليل قبل اللمس (5 دقائق تفكير)

---

```
الجمهور: شباب طموحون → يريدون الإحساس بالقوة والتحدي
الشعور المطلوب: حماس + إلحاح + قوة + إمكانية التغيير
خلاصة الرسالة: "لديك فرصة الآن — لا تضيّعها"
الهوك البصري: صورة لجسم في وسط التمرين (عضلات، حركة، طاقة)
نمط العين: Z-Pattern → الصورة أعلى، العنوان في المنتصف، CTA أسفل يمين
```

---

---

### الخطوة 2: رسم 3 مفاهيم (Thumbnails) — على الورق

---

**المفهوم أ:** صورة كاملة الخلفية + نص فوقها + Overlay غامق
**المفهوم ب:** خلفية سوداء + صورة المنتج/الشخص في الجانب + النص في الجانب الآخر
**المفهوم ج:** تصميم جرافيكي نقي — بدون صورة — اعتماداً على الخطوط والأشكال فقط

**الاختيار:** المفهوم أ — لأن الصورة الإنسانية تُحرّك المشاعر أكثر

---

---

### الخطوة 3: بناء التسلسل البصري

---

```
H1: "أكتوبر = تغيير"  → أكبر + أبيض + خط عريض جداً
H2: "شهر مجاني للأعضاء الجدد" → متوسط + رمادي فاتح
CTA: "اشترك الآن" → زر أحمر + نص أبيض + حجم مناسب
التفاصيل: العنوان + الهاتف → صغير + شفافية 70%
```

---

---

### الخطوة 4: اختيار الألوان

---

**ألوان العميل:** أسود + أحمر
**تطبيق 60-30-10:**

```
60% → أسود #0D0D0D (الخلفية والمساحات الكبيرة)
30% → أبيض/رمادي فاتح #F0F0F0 (النصوص الأساسية)
10% → أحمر #E63946 (الـ CTA فقط + تفاصيل مميزة)
```

**لماذا هذا التوزيع؟**
- لو جعلنا الأحمر 60% → مزعج ومتعب للعين
- لو جعلناه 30% → يفقد قوة التمييز
- عند 10% → كل ما هو أحمر = مهم جداً

---

---

### الخطوة 5: اختيار الخطوط

---

```
العنوان: Bebas Neue أو Black Han Sans (خط كثيف ضخم = قوة)
النصوص: Montserrat Regular (نظيف وحديث)

لماذا Bebas؟ → شخصيته: جرأة، طاقة، رياضة. هو الخط الذي تستخدمه صالات الرياضة والفرق الرياضية.
لماذا Montserrat؟ → مقروء في الأحجام الصغيرة، حديث، يتناسق مع Bebas.
```

---

---

### الخطوة 6: قائمة المراجعة النهائية

---

```
✓ هل الرسالة واضحة في 5 ثوانٍ؟
  → "أكتوبر = تغيير" + شهر مجاني → نعم

✓ هل التسلسل البصري صحيح؟
  → صورة → عنوان ضخم → CTA → تفاصيل → نعم

✓ هل الألوان في حدود 3؟
  → أسود + أبيض + أحمر → نعم

✓ هل الخطوط في حدود 2؟
  → Bebas + Montserrat → نعم

✓ هل المساحة البيضاء كافية؟
  → راجع الهوامش والمسافات → نعم

✓ هل الصورة عالية الجودة؟
  → 300 DPI للطباعة، 1080x1080 للرقمي → نعم
```

### المفهوم الأول: المسارات ونقاط الارتكاز (Paths & Anchor Points)

---

> كل شكل في Illustrator مبني من **نقاط** مربوطة بـ**خطوط أو منحنيات**.

```
نقطة ارتكاز حادة (Corner):    •——•——•
نقطة ارتكاز منحنية (Smooth):  •~~•~~•
```

- انقر بالـ Pen Tool = نقطة حادة
- انقر + اسحب بالـ Pen Tool = نقطة منحنية (Bezier Curve)

### المفهوم الثالث: Pathfinder Panel

---

> أقوى أداة في Illustrator لبناء أشكال معقدة من أشكال بسيطة.

```
اختر شكلَين متداخلَين → افتح Pathfinder Panel → اختر العملية:

UNITE:         A + B = AB (دمج شكلَين في شكل واحد)
┌─────────────────────────────────────────────────────┐
│  ●○ + ●○ = ●●●● (منطقة واحدة موحدة)                │
└─────────────────────────────────────────────────────┘

MINUS FRONT:   B - A = B بدون الجزء المشترك
┌─────────────────────────────────────────────────────┐
│  دائرة صغيرة فوق مستطيل → المستطيل بفتحة دائرية   │
└─────────────────────────────────────────────────────┘

INTERSECT:     احتفظ بالجزء المشترك فقط
EXCLUDE:       احذف الجزء المشترك، احتفظ بالباقي
```

### المفهوم الخامس: Type on a Path

---

> نص يسير على مسار — دائرة، منحنى، أي شكل.

```

اختر Type on a Path Tool (موجود تحت أداة النص)
انقر على حافة الشكل
اكتب النصّ → سيسير على المسار
```

## تحويل النص لمسارات (Outline Text)

---

> **مهم جداً:** قبل تسليم أي ملف Illustrator، حوّل جميع النصوص لمسارات.

**لماذا؟** → لأن الملف على جهاز آخر قد لا يحتوي على نفس الخطوط → النص يتغير شكله.

```
حدد كل النص
Type → Create Outlines (Shift + Ctrl/Cmd + O)
الآن النص أصبح شكلاً → لا يتغير في أي جهاز
⚠️ لكن: بعد هذا لن تستطيع تعديل النص
الحل: احتفظ بنسخة بدون Outline (PSD أو AI قابل للتعديل)
```

## المرحلة ب: رسم الأفكار (Sketching)

---

### الفئات الممكنة للشعار:

| النوع | التعريف | مثال |
|-------|---------|------|
| **Wordmark** | الاسم كاملاً كشعار | Google, FedEx |
| **Lettermark** | أحرف فقط | IBM, HP |
| **Pictorial Mark** | رمز بصري فقط | Apple, Twitter/X |
| **Combination Mark** | رمز + اسم معاً | Starbucks, Adobe |
| **Emblem** | كل شيء داخل شكل | Harley-Davidson |

### مفاهيم SABAH المقترحة:

```
المفهوم 1: Wordmark
"SABAH" بخط هندسي نظيف
الحرف S مُعدَّل ليشبه بخار القهوة

المفهوم 2: Combination Mark
رمز بسيط (قوس = شروق الشمس) + "SABAH" بجانبه

المفهوم 3: Emblem
"SABAH" داخل دائرة + "Since 2024" حول الدائرة
```

**ارسم على الورق — كل مفهوم في 3 دقائق — لا تكمال — مجرد أفكار.**

## المرحلة د: عرض الهوية (Presentation)

---

### الخطوة 1: إعداد Artboards (10 دقائق)

```
File → New Document
→ Web (72ppi) أو Print (300ppi)
→ Multiple Artboards: 3 artboards

Artboard 1: 500 × 500px  (Icon / Stamp)
Artboard 2: 1200 × 400px (Horizontal Wordmark)
Artboard 3: 400 × 1200px (Vertical Variation)

Units: Pixels
Color Mode: RGB للرقمي أو CMYK للطباعة
```

---

### الخطوة 2: بناء الـ Wordmark (25 دقيقة)

**تنفيذ مفهوم SABAH — خطوة بخطوة:**

```
الخطوة أ: كتابة الاسم
- اختر Type Tool (T)
- اكتب "SABAH"
- اختر خطاً هندسياً: Futura أو Montserrat أو Raleway
- الحجم: 120pt للبداية

الخطوة ب: تحويل النص لمسارات
- Shift + Ctrl/Cmd + O
- الآن كل حرف أصبح شكلاً هندسياً

الخطوة ج: Ungroup الحروف
- Object → Ungroup (أو Ctrl+Shift+G)
- الآن كل حرف قابل للتعديل منفرداً

الخطوة د: تعديل الشكل (اختياري - للمتقدمين)
- استخدم Direct Selection (A) لتحريك نقاط فردية
- حاول تعديل حرف S: أضف منحنى خفيف في الأعلى = بخار
- أو: أضف نقطة إضافية + اسحبها = تعديل طفيف

الخطوة هـ: إضافة العنصر الرمزي
- ارسم قوساً بسيطاً (Arc) بأداة Line أو Pen
- ارفعه فوق الحرف S أو قبل الاسم
- Stroke فقط، لا Fill، اللون نفس لون الاسم
```

### الخطوة 4: نسخ الشعار المطلوبة (10 دقائق)

**كل شعار احترافي يحتاج 3 نسخ كحد أدنى:**

```
النسخة 1: Full Color (Positive)
الشعار بألوانه الكاملة على خلفية فاتحة
الاستخدام: المطبوعات، الموقع، الخلفيات الفاتحة

النسخة 2: Reversed (Negative)
الشعار أبيض على خلفية داكنة (أو بالعكس)
الاستخدام: الخلفيات الداكنة، القمصان الداكنة، العروض

النسخة 3: Black (1-Color)
الشعار بالأسود فقط
الاستخدام: الختم، الطباعة أحادية اللون، الفاكس!

النسخة 4: White (1-Color)
الاستخدام: نفس Reversed لكن بدون خلفية
```

> **لماذا هذا مهم؟** الشعار قد يُطبع على قماش، يُوضع على كوب، يُستخدم في فيديو. كل سياق يحتاج نسخة مختلفة.

### الخطوة 5: بطاقة الأعمال في Illustrator (30 دقيقة)

**المقاسات:**
```
حجم البطاقة القياسي: 85 × 55mm
مع bleed: 91 × 61mm (3mm إضافية من كل ناحية)

في Illustrator:
File → New → Custom Size: 91mm × 61mm
الوحدات: Millimeters
Color Mode: CMYK (للطباعة الاحترافية)

ثم أضف guides:
View → Guides → Create Guides
أو اسحب من المسطرة وضع عند 3mm من كل حافة
```

**تصميم الوجه الأمامي (30 دقيقة):**

```

- مستطيل بحجم الـ Artboard (91×61mm)
- Fill: #2C1A0E (Espresso Brown)
- لا Stroke

الشعار:
- ضع نسخة الشعار White (الأبيض)
- مركّز أفقياً وعمودياً
- حجم مناسب — لا يجب أن يملأ كل المساحة

عنصر ذهبي اختياري:
- خط رفيع 0.5pt بلون #C9A84C
- في أسفل البطاقة كـ separator

المساحة البيضاء:
- الشعار يأخذ 40-50% من البطاقة
- باقي المساحة = فراغ مقصود = يوحي بالفخامة
```
